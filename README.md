# ScummVM GTK

ScummVM game launcher with grid view and game details.

Built with GTK4/Adwaita. Part of the [Danne L10n Suite](https://github.com/yeager/debian-repo).

## Installation

### Debian/Ubuntu
```bash
sudo apt install scummvm-gtk
```

### Fedora/RPM
```bash
sudo dnf install scummvm-gtk
```
